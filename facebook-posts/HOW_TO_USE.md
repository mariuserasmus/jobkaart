## Step 2: Take a Screenshot

### On Windows:
1. Open the HTML file in your browser
2. Press **F11** to go fullscreen (hides browser bars)
3. Press **Windows Key + Shift + S** (Snipping Tool)
4. Drag to select the entire design
5. The screenshot is automatically saved to your clipboard
6. Open Paint or Photos and paste (Ctrl+V)
7. Save as PNG or JPG

### Alternative (Easier):
1. Open the HTML file
2. Right-click on the page
3. Select "Save as PDF" or "Print"
4. Choose "Save as PDF"
5. Then use any PDF to Image converter online

### On Mac:
1. Open the HTML file
2. Press **Cmd + Shift + 4**
3. Drag to select the design
4. Screenshot saves to Desktop automatically

## Which Design Should You Use?

### **Design 1: Lost Money (Purple Gradient)**
**Best for:** Grabbing attention with the pain point
**Use when:** Posting in general tradie groups
**Caption style:** Question format ("How many quotes did you forget?")

### **Design 2: Before vs After (Split Screen)**

## Pro Tips for Maximum Reach:

### 1. **Post at Peak Times**
- **Best times for SA tradies:**
  - Tuesday-Thursday: 7-8am (before work starts)
  - Tuesday-Thursday: 6-8pm (after work, scrolling time)
- **Worst times:**
  - Weekends (people aren't thinking about work)
  - Monday mornings (too busy)

### 2. **Use the Right Groups**
Post in groups like:
- "South African Plumbers"
- "Electricians South Africa"
- "SA Small Business Owners"
- "Tradie Talk SA"
- "Handyman Services South Africa"

### 3. **Don't Spam**
- Post in max 3-4 groups per day
- Space them out (morning, lunch, evening)
- Use different captions for each group
- Wait 2-3 days before posting again

### 4. **Engage FAST**
- When someone comments, reply within 5 minutes
- Facebook sees engagement and shows your post to more people
- Every comment = more reach

### 5. **Track What Works**
After each post, check Google Analytics:
1. Go to analytics.google.com
2. Reports ‚Üí Realtime
3. See if people are clicking through

Make a note:
- Design 1 + Question format = 12 clicks
- Design 2 + Story format = 3 clicks
- Design 3 + Value format = 18 clicks

**Double down on what works!**

## If You Want to Edit the Designs:

### Change the Text:
1. Right-click the HTML file ‚Üí Open with Notepad (or VS Code)
2. Find the text you want to change (it's in plain English, easy to find)
3. Change it
4. Save the file
5. Refresh your browser

### Change the Colors:
Look for lines like:
- `background: #2563eb;` (blue)
- `color: #dc2626;` (red)

Use a color picker: https://htmlcolorcodes.com/

### Change the Size:
Look for:
- `font-size: 48px;` and change the number
