7. WhatsApp Integration
Critical for SA market: WhatsApp is how tradies communicate with customers.
Approach: Click-to-Chat (No API Pain)
Instead of complex WhatsApp Business API integration, we use smart linking:
How it works:
User taps "Send via WhatsApp"
App generates public link to quote/invoice (e.g., jobkaart.co.za/q/abc123)
WhatsApp opens with pre-filled message containing link
User taps Send in WhatsApp
When customer clicks link, JobKaart tracks: "Quote viewed at 14:32"
Benefits of This Approach
