# Afrikaans Touches Added to Landing Page

**Date**: November 29, 2025
**Purpose**: Show bilingual support (English/Afrikaans) for SA tradespeople

## Why Afrikaans Matters

### Target Market Reality
- Many SA tradies are Afrikaans-speaking
- "JobKaart" itself is an Afrikaans word (Job Card)
- Shows cultural understanding and local focus
- Differentiates from international competitors

### Approach
- **Subtle, not overwhelming**
- English first, Afrikaans as supportive accent
- Italicized/secondary styling
- Reinforces key messages in both languages

---

## Translation Notes

### Key Terms

| English | Afrikaans | Notes |
|---------|-----------|-------|
| Stop losing jobs | Moenie weer werk verloor nie | Idiomatic, natural phrasing |
| Get paid faster | Kry vinniger betaling | Direct and clear |
| Things | Dinge | Simple, everyday word |
| Really well | Regtig goed | Colloquial emphasis |
| The maths | Die somme | "Somme" = sums/calculations |
| Matters | Saak maak | "Makes a difference" |
| Built for | Gebou vir | Strong, intentional |
| SA tradespeople | Suid-Afrikaanse ambagsmense | Formal but accessible |

### Translation Philosophy

1. **Colloquial over formal** - How tradies actually speak
2. **Accessible** - No complex vocabulary
3. **Natural flow** - Sounds authentic, not Google Translated
4. **Parallel structure** - Matches English rhythm where possible

---

## Visual Hierarchy

### Styling Approach

**English Text**:
- Primary font size
- Bold or regular weight
- Main color (white, gray-900, etc.)

**Afrikaans Text**:
- Slightly smaller font size
- Italic
- Lighter color (blue-200, gray-500, gray-400)
- Secondary/supportive position

**Why this works**:
- English speakers can ignore Afrikaans (doesn't clutter)
- Afrikaans speakers see their language (feels inclusive)
- Clear visual hierarchy (English primary, Afrikaans accent)

## Strategic Placement

### Where We Added Afrikaans ‚úÖ
1. **Hero** - First impression (bilingual welcome)
2. **Features** - Core value proposition
3. **ROI** - Financial justification
4. **Footer** - Final local touch

### Where We DIDN'T Add Afrikaans ‚ùå
1. **Form labels** - Keep simple (English standard for forms)
2. **Button text** - Action words best in one language
3. **Privacy notices** - Legal clarity requires English
4. **Feature descriptions** - Too much text, would clutter

**Principle**: Afrikaans as accent, not translation

## Cultural Authenticity

### Why It Feels Authentic

1. **"JobKaart" brand name** - Already Afrikaans
2. **Strategic placement** - Key messages only
3. **Natural phrasing** - How tradies actually talk
4. **Subtle styling** - Not forcing bilingualism
5. **Local references** - "Suid-Afrikaanse" (not "South African" translated)

### Avoiding Tokenism

**Don't**:
- ‚ùå Translate everything (feels forced)
- ‚ùå Use formal Afrikaans (sounds corporate)
- ‚ùå Mix languages mid-sentence (confusing)

**Do**:
- ‚úÖ Use at key emotional moments
- ‚úÖ Keep it colloquial and natural
- ‚úÖ Let English be primary (wider audience)
- ‚úÖ Afrikaans as welcoming accent
